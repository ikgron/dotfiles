# Dotfiles

![screenshot](assets/terminal_screenshot.png)

This is my macOS configuration.
It contains setup and configurations for homebrew, iTerm2, bash, macOS defaults, and more.
I have also set this up to work with Linux, although it doesn't contain as much.

# Installation

**Warning:** Use at your own risk! These dotfiles don't include anything crazy but may cause issues or change stuff you didn't want changed. Please review what will run before bootstrapping!

<br />

Clone the repository wherever you want, it will work from anywhere:

```bash
git clone https://codeberg.org/parser/dotfiles.git
cd dotfiles
```

To run:

```bash
bash ./bootstrap.sh
# or `chmod +x ./bootstrap.sh && ./bootstrap.sh`
```
